KRepoXplorer

A cross-platform desktop Git repository manager. Scan a folder, browse all your repos, and inspect branches, commits, and diffs — all from one window.

↓ Download Latest Release ⑂ View on GitHub

Features

📁

Repository Browser

Add one or more base folders. KRepoXplorer discovers all Git repos inside them automatically.

🌿

Branches

List all local and remote branches for the selected repository at a glance.

📜

Commit History

Browse recent commits with author, date, and commit message.

🔍

Working-Tree Changes

See modified, added, and deleted files with per-file diffs.

↔

Side-by-Side Diff

Two-column view pairing removed and added lines with per-token syntax highlighting.

🎨

Syntax Highlighting

19 languages highlighted in diffs: Dart, JS/TS, Python, Rust, Go, PHP, and more.

⚙️

Settings

Toggle between unified and side-by-side diff modes. Preferences are persisted across sessions.

🔄

Auto-Refresh

Optional 5-second polling keeps the view up to date without manual refreshing.
